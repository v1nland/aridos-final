Cuentas

En esta sección se encuentra la configuración global de cada cuenta asociada en simple.

Configuración de Agenda

La configuración de la instancia de agenda es única para todas las cuentas en SIMPLE, compartiendo un appKey que se configura en el archivo config.php y requiere una instalación del microservicio de agendas (documentación en swagger), servicio: apps, método POST: {{protocol}}://{{server}}:{{port}}/{{context}}/apps.

Así, cada aplicación, es una cuenta en simple, asociada una misma instancia (appkey). La creación de cuenta en la API requiere también información de configuración de contenido de correos y su origen.

Una vez se ha creado la llave de instalación (appKey), cada cuenta debe tener un nuevo dominio y estar creada en la API de agendas. Para ver la configuración de la cuenta en SIMPLE:

Ingresar a la opción de cuentas del menú izquierdo.

Editar una cuenta particular

En la sección "Editar configuración de Agenda" se puede consultar la configuración que corresponde a la instancia de agenda.

  1. Clave App. Es un valor de solo lectura, que se extrae de config.php.
  2. Dominio. Corresponde al identificador único de la cuenta para la correspondiente aplicación en la API de agendas.
